- Windows 下常用的是微软开发的 Visual C++（MSVC）、Clang、CMake 或 MS Build；

> Visual C++（简称 MSVC）是由微软开发的，只能用于 Windows 操作系统；GCC 和 LLVM Clang 除了可用于 Windows 操作系统之外，主要用于 Unix/Linux 操作系统。
